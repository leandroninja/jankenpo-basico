# Jankenpo ✊✋✌️

Jogo de Pedra, Papel e Tesoura moderno, responsivo e divertido — direto no navegador, sem dependências.

## Como jogar

1. Abra o arquivo `index.html` no navegador
2. Escolha **Pedra**, **Papel** ou **Tesoura**
3. O computador faz sua jogada automaticamente
4. Acompanhe o placar e o histórico de rodadas

## Funcionalidades

- Tema escuro moderno com gradientes e partículas animadas
- Emojis grandes e animados para cada jogada
- Animação de "CPU pensando" antes de revelar a escolha
- Placar com contagem de vitórias do jogador e do computador
- Badge de sequência de vitórias (🔥 streak)
- Mensagens de provocação aleatórias a cada rodada
- Confetes coloridos ao vencer
- Sons via Web Audio API (vitória, derrota, empate)
- Histórico completo das rodadas com resultado colorido
- Layout responsivo — funciona em desktop e mobile
- Botão de reiniciar para zerar o placar

## Tecnologias

- HTML5
- CSS3 (animações, variáveis, grid, flexbox)
- JavaScript (ES6+, Web Audio API)
